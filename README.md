# Photo Tools WebApp

This is a mobile-friendly web application with 3 main tools:

1. **Image Resize** - Resize any image to custom width/height.
2. **Background Remove** - Remove background (simple demo version).
3. **Image Compress** - Compress image to reduce size.

## How to Use
- Open `index.html` in browser.
- Choose the tool you want.
- Upload image and click action button.
- Download the result.
